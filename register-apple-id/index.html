1. 获取账号

在获取之前，需要准备以下几个东西

  • 能接收短信的手机号（可以是国内的）
  • 一个未获取过 Apple ID 的邮箱（QQ 邮箱不要用）

进入 Apple ID 获取页面：https://appleid.apple.com/account，填写图示中相应信息。

之后验证电子邮件和手机号码，将收到的验证码填入图示中位置

获取成功后，账号会自动登录，如果没登录的话重新登录一遍刚获取的账号即可。然后点击「个人信息 」，接着再点击「 国家或地区」

选择「更改国家或地区」

之后会跳转更改付款方式的页面。

2. 绑定支付方式

在添加付款方式的页面里，把 国家或地区更改成「***美国***」，付款方式按标注填写

WildCard 申请参考：推荐虚拟卡，使用支付宝购买开卡

美国电话是必须要填写的，这个号码不用验证，您可以随便填一个，美国电话一共 10 位，前 3 位的区号一定要对，建议填跟账单地址同一地区的区号：

  • Delaware 302
  • Montana 406
  • Oregon 541
  • New Hampshire 603

后面 7 位可以随便填

也可以访问这个网站生成号码：https://www.generatormix.com/random-phone-numbers，复制一个随机生成一个手机号即可。

点击更新 Save Changes，这个时候 Apple ID 已经更换到美区

接着点击 *「 *****付款与配送******」**，卡片已经添加成功，不过还需要再次验证下卡片信息，点击卡片输入 CVV（WildCard 卡片界面有，复制即可），点击存储更改

到这里美区 Apple ID 就获取完成了，之后就可以在 Apple Store 登录刚刚获取的美区 Apple ID 来订阅自己需要的软件了
